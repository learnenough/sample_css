Important HTML tags

This table doesn't include all HTML tags, or even most of them, but it does list some of the most important ones.

Tag Name Purpose Example Result
em emphasis emphasize text <em>technical sophistication</em> technical sophistication
strong strong make text stronger <strong>at least a billion</strong> people at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough™</a> Learn Enough™
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl" />
ol ordered list list elements in numerical order
ul unordered list list elements whose order doesn't matter
li list item include a list item (ordered or unordered)
table table include a table
tr table row include a row of data
th table heading make a table heading
td table data include a table data cell
